Welcome to "fast" pandas
========================

**fastpandas** is an odd collection of tips for using *pandas* and other related tools effectively. It is inspired by documents like `R Inferno`_ and inherits some of its idiosyncrasies as a result.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.

.. _`R Inferno`: https://www.burns-stat.com/pages/Tutor/R_inferno.pdf
