Welcome to "fast" pandas
========================

**fastpandas** is an odd collection of tips for using ``pandas`` and other related tools effectively. It is inspired by `R Inferno`_ and inherits some of its idiosyncrasies.

.. warning::

   Please beware that all the content presented is a model of the truth as opposed to the truth itself. Concretely, this means that I've used terminology and concepts in a way that I find useful, not necessarily the most technically accurate usage of the terms. That said, I always welcome feedback on how the material can be improved.

.. _`R Inferno`: https://www.burns-stat.com/pages/Tutor/R_inferno.pdf
